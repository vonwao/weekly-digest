AGENTS.md Outperforms Skills in Agent Evals

Vercel Blog

Vercel's research on teaching coding agents:

  • Skills (invocable docs packages): 53% pass rate, same as no docs
  • AGENTS.md (8KB compressed docs in root): 100% pass rate
  • Skills weren't triggered reliably (56% never invoked)

Takeaway: Put framework docs directly in AGENTS.md, don't rely on agents to invoke skills.
